# eIncu mobile 
This is a user friendly and well designed app for farmers
## Purpose 
***eIncu*** mobile has the purpose of allowing farmers to remotely control the encubator from any where. 
--- 
It displays analytics via the use modern graphs and shows the time remaining for the eggs to hatch. 
--- 
The ***eIncu*** is able to allow farmer to choose which egg types it is going to hatch. 
This makes ***eIncu*** a solution as existing encubators are used for only a single category of eggs like for chicken eggs not duck eggs.
