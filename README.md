# eIncu – AI/IoT Smart Chicken Incubator

## Overview
eIncu is an AI- and IoT-driven chicken incubator designed to optimize hatching conditions by automatically controlling temperature, humidity, and monitoring environmental parameters in real-time.

## Objectives
- Maintain optimal incubation conditions
- Reduce manual intervention
- Improve hatch rate using data-driven insights

## Features
- Real-time temperature and humidity monitoring
- Automated control (heater, fan, humidifier)
- Remote monitoring via web/mobile interface
- Data logging for analysis
- (Planned) AI-based optimization / anomaly detection

## System Architecture
- Microcontroller (e.g., Raspberry pi/ESP32 / Arduino)
- Sensors (DHT11/DHT22, etc.)
- Actuators (heater, fan, relay)
- Backend server (Node.js / Python)
- Dashboard (web app)

## Research Component
This project explores:
- IoT-based environmental control systems
- Predictive modeling for incubation optimization
- Fault detection in embedded systems
